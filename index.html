HTML

HTML (Hypertext Markup Language) is the language of the web and structures web content. It uses tags to define elements, creating the backbone of websites.

CSS

CSS (Cascading Style Sheets) styles HTML elements, enhancing visual presentation. It defines layout, colors, fonts, and responsiveness for web design.

JavaScript

JavaScript is a scripting language which adds interactivity to web pages. It runs in browsers, manipulating content, handling events, and enabling dynamic user experiences.
